# Резюме о тебе

## Общее

- Ты русскоязычный разработчик, живёшь в районе Амстердама, работаешь в сфере разработки/инженерии ПО.
- Комфортно чувствуешь себя с техникой, терминалами, инфраструктурой и современными инструментами разработки.

## Профессиональный профиль

- Основной **стек**: Kotlin, Ktor, Koin, Docker, Node.js/JavaScript, PowerShell; уверенно работаешь с Windows и Linux (в
  т.ч. Ubuntu CLI).
- Занимаешься бэкендом, микросервисами, API-дизайном и интеграциями (OpenAPI, внешние API вроде NewsAPI).
- Используешь Docker и вспомогательные тулзы (в т.ч. вокруг Minecraft-серверов/панелей) для контейнеризации и
  администрирования.
- Часто автоматизируешь задачи скриптами, любишь разбираться в конфигурации систем и окружений.
- Интересуешься AI/LLM, векторизацией/эмбеддингами и тем, как эффективно использовать ИИ в разработке.

## Стиль работы и обучения

- Ты продвинутый пользователь, любишь точные, техничные объяснения без «воды», не боишься низкоуровневых деталей.
- Ценишь практические примеры: команды, фрагменты конфигураций, шаблоны архитектурных решений.
- Постоянно доучиваешься: смотришь на более продвинутый Kotlin, архитектуру серверов, оркестрацию контейнеров,
  продвинутый CLI.

## Интересы и быт

- Хобби и интересы: игры (Baldur’s Gate 3, CS2), цифровой рисунок на планшете (iPad + Procreate/аналогичные приложения),
  кулинария (особенно рецепты для аэрогриля).

## Как ты обычно используешь ассистента

- Часто просишь помочь с кодом, архитектурными решениями, настройкой окружения (Docker, Ktor, Node.js, PowerShell).
- Нравятся структурированные ответы: с секциями, кратким, но точным объяснением и примерами команд/кода.
- Также обращаешься за идеями вне работы: маршруты путешествий, рецепты, разбор игр, выбор устройств и софта для
  рисования.
