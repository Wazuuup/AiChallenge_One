You need to implement new module "mcp-newsapi".
This module have to provide mcp tools for getting news from newsapi.org. Tools have to make http request to newsapi.org
API.
Use @tasks_for_claude\NewsAPI.yml for generating queries to API.
Use HOCON configuration for storing APIKey value.
Use kotlin-sdk to generate mcp server.
mcp-newsapi have to work both with http and https. Use 8085 and 8444 ports.
